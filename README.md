# Document Versioning App
Full-stack app for controlled document versioning with dual-LLM validation (Executor + Validator) and SMR compliance.
## Quick Start
1. Copy `.env.example` to `.env` and set `OPENROUTER_API_KEY` and `SECRET_KEY`
2. Run `docker-compose up --build -d`
3. Access frontend at http://localhost:3000
4. Login: admin@example.com / admin123
